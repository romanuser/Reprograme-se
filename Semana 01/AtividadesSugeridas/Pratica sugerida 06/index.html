Pratica Sugerida Aula 06

Semana 01

Aula 01 - Como Funcionam os Sites

Nessa videoaula, é mostrado como funcionam os sites nos bastidores, que tipo de informação trafega entre o navegador do usuário e o servidor web, tudo de uma maneira bem simplificada e didática.

Semana 02

Aula 02 - A Estrutura Básica de Arquivos HTML e CSS

Nesta videoaula, é mostrado como é a estrutura básica de um documento HTML, explicando o significado e a utilidade de cada elemento. Explica-se também como é a estrutura dos arquivos do tipo CSS, dos seletores e blocos de propriedades.

Semana 03

Aula 03 - Editores de Código Repl.it e VSCode

Nesta videoaula, apresenta-se os ambientes de desenvolvimento que utilizaremos ao longo do curso. O primeiro deles é o editor online Repl.it, que permite editar e executar códigos-fontes em mais de 50 linguagens de programação, incluindo HTML, CSS e Javascript. Em seguida,é apresentado o Visual Studio Code, um editor de código gratuito e o mais popular da atualidade.

Semana 04

Aula 04 - Elementos HTML Básicos para Exibição de Textos

Nesta videoaula, é mostrado os elementos HTML textuais básicos, incluindo elementos para títulos e subtítulos, parágrafos, listas, links, apresentação de código-fonte, entre outros. Além de mostrar como utilizar cada tipo de elemento, também é dito a respeito da questão semântica de cada elemento.

Semana 05

Aula 05 - Elementos HTML para Exibição de Imagens

Nessa videoaula, é mostrado os elementos HTML usados para se exibir imagens, principalmente o elemento "img". É mostrado também as diferenças entre imagens comuns e vetoriais, as formas de se mapear uma imagem em regiões clicáveis usando o elemento "map", e um recurso que permite carregar imagens específicas para determinadas resoluções de tela usando o elemento "picture".

Semana 06

Aula 06 - Elementos HTML para Exibição de Conteúdo Multimídia

Nesta videoaula, é apresentado os elementos HTML usados para se exibir conteúdos multimídia, a saber, áudio e vídeo. É mostrado também como exibir vídeos e áudios a partir de arquivos com formatos compatíveis com os navegadores atuais. Na parte dos vídeos, é ensinado como criar, editar e exibir legendas para os vídeos disponibilizados. Também é mostrado como incorporar vídeos e playlists do YouTube em uma página HTML.

Semana 07

Aula 07 - Criação de Imagens SVG usando HTML

Nessa videoaula, é mostrado os elementos HTML usados para se criar imagens vetoriais do tipo SVG, passando por todas as formas básicas e seus atributos e mostrando também como criar animações baseadas em elementos e atributos de elementos.
